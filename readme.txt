== Description ==

An alternative "Edit Page" list design. Makes it easier to get an overview of the page list. Just activate, nothing else.

== Installation ==

Easy installation:<br />
1. Upload `edit-page-list-design.php` to the `/wp-content/plugins/` directory<br />
2. Activate the plugin through the 'Plugins' menu in WordPress<br />
<br />
Yes, thats it!
